# JScon

Helps To fetches javascript files from a list of URLS or subdomains

## Resources

* [Usage](#Usage)
* [Installation](#Installation)

## Usage:

Examples:

```
$ cat targets.txt
abc.com
xyz.com
example.com
```

```
$ chmod +x JScon.sh
$ sudo ./JScon.sh targets.txt
```
